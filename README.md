# Tab Zen - Chrome Extension

Tab Zen is a minimalist Chrome extension to help you manage your browser tab chaos. Save sessions, group by domain, and close duplicates with one click.

## Features

- **Save Sessions**: Capture all your open tabs as a named session for later restoration
- **Restore Sessions**: Reopen all tabs from a saved session with one click
- **Close Individual Tabs**: Remove specific tabs from saved sessions
- **Close All Tabs**: Close all tabs belonging to a session at once
- **Search Sessions**: Find sessions by name, tab title, or URL
- **Session Management**: Rename or delete saved sessions as needed
- Built with React + TypeScript + Vite

## Installation (Dev)

1. Clone this repo
2. Run `npm install`
3. Build with `npm run build`
4. Go to `chrome://extensions`, click "Load unpacked", and select the `dist/` folder.
